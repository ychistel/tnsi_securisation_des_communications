Sécurisation des communications
===============================


Premier système de cryptographie utilisé par les romains pour protéger leurs messages. Ce chiffrement rudimentaire est un chiffrement dit symétrique puisque la clef utilisé pour chiffrer et déchiffrer le message est la même.
